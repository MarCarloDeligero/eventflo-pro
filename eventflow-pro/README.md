# EventFlow Pro (starter scaffold)

This is a minimal starter project scaffold generated for you.

## Structure
- index.php - front controller
- includes/ - config and helpers
- classes/ - PHP classes (Database, Event)
- templates/ - header/footer
- pages/ - home and event pages
- admin/ - simple admin index
- api/ - example JSON endpoint
- assets/ - css, js, images

## Setup
1. Create a MySQL database and update includes/config.php with credentials.
2. Create an `events` table (id, title, date, location, description).
3. Place the project in a PHP-enabled server root.
